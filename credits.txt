fluorescence makes extensive use of ClanLib (www.ClanLib.org). 

The "Devinne Swash" font was created by Dieter Steffmann (http://www.steffmann.de/). It is available at http://www.fontspace.com/dieter-steffmann/devinne-swash

fluorescence is based on pugixml library (http://pugixml.org)
pugixml is Copyright (C) 2006-2010 Arseny Kapoulkine.

fluorescence makes extensive use of several boost libraries (www.boost.org). 

fluorescence relies on the International Components for Unicode (http://site.icu-project.org/).

Sound in fluorescence make possible by FMOD (www.fmod.org).

The explosion.png and firesmoke.png effect textures were created by the Iris2 Team/BuD (www.iris2.de).

The fluorescence updater is based on subversion (http://subversion.apache.org/), the win32svn build (http://sourceforge.net/projects/win32svn/) and svncpp (http://rapidsvn.tigris.org/svncpp.html). 
Some code snippets for the updater are originally by the Iris2 Team (www.iris2.de).

fluorescence uses zlib (www.zlib.org) to inflate compressed data.

The button textures used in the default theme were created by Blarumyrran (http://opengameart.org/content/pixelart-menu-naturery-hand-shooting-some-green-stuff).
